# списки

чтобы добавить ненумерованные списки, необходимо пункты выделить зведочкой.  
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить номированные списки, необходимо пункты просто пронумеровать.   
Напрмер, вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3
